# Lumi Social Intelligence v0.4.2 care-release evidence

**Status:** `verified`<br>
**Version:** `0.4.2`<br>
**Principle:** Live Surface reduces uncertainty before it reduces task time<br>
**Claim boundary:** Public-safe care contract/evidence for instant Live Surface reactions and Next-Step Care; no public repo Telegram sends

v0.4.2 records the missed care requirement from the original Live Surface release: tiny safe actions must not make the user watch visible tool spelunking. For an already-warmed reaction target, the contract is **0 tool calls**, no tool discovery, no DB archaeology, and no guessing.

## Next-Step Care

Next-Step Care is procedural warmth: tiny predictable narration before side effects, especially where uncertainty would create cognitive load.

- Before action: “I’m going to react with ❤️ to your latest Telegram message.”
- After success: “Done ❤️”
- Fail closed: “I can’t safely find the last message id, so I won’t guess.”
- Accessibility note: Care infrastructure for neurodivergent users; not cosmetic polish
