# Lumi Social Intelligence v0.3.0

Release-candidate notes for **Lumi Social Intelligence v0.3.0**: shadow-only Live Surface natural-language controls.

## Observed / verified in this repository

The v0.3.0 development surface verifies a public-safe control path:

```text
ordinary user phrasing → intent family → reviewable control state → tiny acknowledgement
```

Included surface:

- `lumi_social_intelligence/live_surface_controls.py`
- `docs/live-surface-natural-language-controls.md`
- `tests/test_live_surface_controls.py`

The tests verify safe local controls, Calendar/free-busy gating, external-send blocking, scheduler/config blocking, durable-memory review gates, no raw text storage, no runtime-plumbing acknowledgements, and zero side-effect counters.

## Review-gated / no-write boundary

This release-candidate remains non-mutating:

```text
telegram_messages_sent: 0
telegram_reactions_sent: 0
external_writes: 0
calendar_reads_without_surface: 0
email_reads: 0
canonical_memory_writes: 0
runtime_config_writes: 0
scheduler_mutations: 0
personal_data_reads: 0
```

A request such as “check my calendar and keep plans fresh” is recognized as a personal-data surface request, but it produces a confirmation-gated acknowledgement instead of reading Calendar.

## Not yet supported or not yet claimed live

| Behavior | v0.3.0 RC status |
|---|---|
| Natural-language Live Surface control parsing | observed in this repository |
| Safe local/session control acknowledgement | observed in this repository |
| Personal-data surface expansion | review-gated, not applied |
| Calendar/email reads | not supported by this release-candidate |
| External sends/posts/replies | blocked in shadow mode |
| Runtime config or scheduler edits | blocked in shadow mode |
| Durable memory promotion | review-gated, not applied |

Before considering a v0.3.0 tag/release, run:

```bash
python3 -m pytest -q tests/test_live_surface_controls.py
./scripts/release_check.sh
python3 scripts/build_release_artifacts.py --output-dir /tmp/lumi-v0.3.0-artifacts
python3 scripts/prepare_release_candidate.py --tag v0.3.0 --report /tmp/lumi-v0.3.0-release-plan.json
```

Publishing remains a human decision. The release-candidate is built to show the Live Surface control boundary honestly: natural, low-friction controls where safe; review gates where personal data, memory, scheduler, runtime, or external-send authority would expand.
