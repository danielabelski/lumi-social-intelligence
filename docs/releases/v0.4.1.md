# Lumi Social Intelligence v0.4.1

v0.4.1 is a patch release for the Live Surface boundary around native Telegram reactions. The live Hermes runtime path has verified native Telegram emoji reaction delivery, and this public doorway now packages the public-safe contract and evidence for that result.

## Release notes

The important adapter contract is narrow: Telegram native reactions through python-telegram-bot use `Bot.set_message_reaction` with a list containing `ReactionTypeEmoji(emoji='❤️')`. A primitive emoji string is not the release contract.

This release does not ship the private Hermes adapter, raw private Hermes runtime state, secrets, logs, chat history, or credentials. It documents the verified boundary and keeps the public repo side-effect-free.

## Added

- Public-safe `v0.4.1` native reaction evidence.
- Release artifact manifest coverage for the verified Telegram reaction payload contract.
- Release notes that distinguish the public evidence boundary from the private Hermes adapter implementation.

## Claim boundary

Hermes native Telegram emoji reaction delivery is verified through the private runtime adapter. Lumi Social Intelligence `v0.4.1` packages only the contract and public-safe evidence.
