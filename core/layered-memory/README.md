# Lumi Layered Memory release surface

Promote tested, public-safe **Lumi Layered Memory** code, examples, and documentation here from the private Lumi Layered Memory development repository.

This directory is a curated release surface, not a raw development dump.

Promotion into this directory is gated by `docs/module-release-gates.json` and
`scripts/validate_module_release_gates.py`. The package name reserved for the
public release surface is `lumi-layered-memory`; private development package
names and raw runtime artifacts must not be copied here.
