# Presence release surface

Promote tested, public-safe **Presence** code, examples, and documentation here from the private Presence development repository.

This directory is a curated release surface for governed initiative, restraint, repair, and public-safe examples.

Promotion into this directory is gated by `docs/module-release-gates.json` and
`scripts/validate_module_release_gates.py`. The package name reserved for the
public release surface is `lumi-presence`; private development package names and
raw runtime artifacts must not be copied here.
