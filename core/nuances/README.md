# Nuances release surface

Promote tested, public-safe **Nuances** code, examples, and documentation here from the private Nuances development repository.

This directory is a curated release surface for contextual appraisal, review cards, and public-safe examples.

Promotion into this directory is gated by `docs/module-release-gates.json` and
`scripts/validate_module_release_gates.py`. The package name reserved for the
public release surface is `lumi-nuances`; private development package names and
raw runtime artifacts must not be copied here.
